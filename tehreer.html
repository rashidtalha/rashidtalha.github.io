Tehreer (تحریر)

I sometimes (very rarely) need to write in Urdu. One possibility is to use a WYSIWYG word processor like Google Docs or MS Word. But MS Word is expensive, and Google Docs doesn't support any decent Urdu font.

Another option is to use Latex (as I usually do for most other things). But, that requires writing Urdu in a code editor, and most of the popular ones have very poor support for RTL languages. This is particularly noticable in Sublime Text. So, one solution was to write all the Urdu text in an HTML-based textbox, where it is easy to edit and then copy it over into a latex file and compile. This is not ideal, but significantly better than directly trying to write Urdu without proper RTL support or decent fonts.

This "app" is desingned to be simple and lightweight.
